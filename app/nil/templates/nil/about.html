  • ABOUT

    monq is a simple web app to make snippet writing easy. It is designed and implemented by @sandcoder because he felt that writing code snippets should be awesome :)

  • FAQ

  • What does this site do?

    This website allows people (like you or me) to create and store Snippets of code (or any text really).

  • Isn't this just a pastebin?

    Well yea, but it aims to make the whole pastebin experience a bit more pleasant.

  • What languages/syntax do you support?

    Because monq uses the wonderful Ace Editor, any language supported by Ace will be available here (NOTE: We may limit the list of supported languages in the future, so just watch out for those changes).

  • How can I make an account?

    Just go < here >. We currently support social logins for now.

  • What is social login?

    Social login is when you use your login information from a social network (Facebook, Twitter) to make an account on an external website. If you have an account on our supported social networks, then you can make an account here!

  • I have multiple social accounts. Can I link them into one account?

    Nope. Only one account per social account (for now at least).

  • What social networks can I log in from?

    We currently support Twitter, Facebook, and Google.

  • Can I trust this site?

    Of course!.....can I trust you?

  • What technologies did you use to make this web app?

    • Django - The awesome Python web framework.

    • Ace Editor - Embeddable Javascript code editor.

    • Twitter Bootstrap - Great frontend framework for your app.

    • Font Awesome - Wide selection of great font icons.

    • highlight.js - Syntax highlighting done right.

    • underscore - Handy collection of functions.

    • jQuery - always
